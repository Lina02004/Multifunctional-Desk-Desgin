# Candy-Themed Multifunctional Desk (E-Design 100)

## Overview

This project is a multifunctional desk designed as part of my E-Design 100 cornerstone engineering course. The assignment focused on user-centered design and required the creation of an original CAD assembly with multiple unique parts.

The desk combines functionality with a playful, candy-inspired aesthetic, featuring a pink-and-white color scheme with decorative elements such as donuts and a candy cane theme. The goal was to design a piece that is both practical and visually expressive.

## Design Concept

The desk was designed to serve multiple use cases in a compact and flexible form:

* A bedside or over-bed desk that can be moved into position
* A flat work surface for general use
* An adjustable drawing table (similar to an architectural drafting table)

The adjustable surface allows the user to change the angle for drawing, writing, or working, while the height can also be modified for comfort and ergonomics.

## Key Features

* Adjustable tabletop (flat or angled for drawing)
* Height-adjustable structure for ergonomic use
* Integrated light box for sketching and tracing
* Built-in pen holder for convenience
* Attached lamp for improved visibility
* Mobile design with simplified wheel components (conceptualized for carpet use)

## CAD & Engineering Focus

This project emphasized CAD modeling and assembly design rather than full mechanical detailing. As part of the course requirements:

* The design includes more than five original parts
* Components were created from scratch rather than reused
* The focus was on structure, layout, and assembly relationships

Some elements (such as the wheels) were intentionally simplified to prioritize the overall design and meet project scope constraints.

## Design Reflection

A major focus of this project was the visual and creative aspect of design. While the desk is functional and ergonomic, more emphasis was placed on aesthetics and customization than on detailed mechanical engineering.

Future iterations could further refine:

* Mechanical realism of moving components
* Structural detailing and material considerations
* Functional optimization of adjustable features

## Tools Used

This project was completed as part of E-Design 100, a cornerstone course introducing engineering design, CAD modeling, and user-focused problem solving.
